# SwifterX iOS (SwiftUI Prototype)

This folder contains the SwiftUI source scaffold for the UI-first iOS prototype based on the provided Figma design.

## Structure
- `App/`: app entry and root tab shell
- `DesignSystem/`: tokens and reusable components
- `Features/`: Home, Cart, Profile, and shared mock models/data
- `Resources/Assets.xcassets/`: asset catalog root
- `Docs/`: visual and interaction validation checklist

## Next Step in Xcode
1. Create a new iOS App target named `SwifterX` (SwiftUI, iOS 17+).
2. Add all files in this folder to the Xcode target.
3. Set `SwifterXApp` as the app entry point if needed.
4. Run on iPhone SE and iPhone 15/16 simulators for parity checks.
